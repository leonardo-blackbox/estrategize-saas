1. **Rewrite AdminIAPage.tsx** (max 200 lines) as the aggregator:

   Imports:
   - `{ useState }` from react
   - `{ motion }` from framer-motion
   - `{ useQuery, useMutation, useQueryClient }` from @tanstack/react-query
   - `{ staggerContainer, staggerItem }` from `../../../../lib/motion.ts`
   - `{ adminListDocuments, adminUploadDocument, adminDeleteDocument, adminTestQuery }` from `../../../../api/knowledge.ts`
   - `{ KnowledgeTestResult }` from `../../../../types/knowledge.ts`
   - DocumentUploadArea, DocumentList, TestQueryPanel from local files (relative `./ ` imports)

   State:
   - `deletingId: string | null` for tracking which document is being deleted
   - `testResult: KnowledgeTestResult | null` for test query response
   - `testError: string | null` for test query error

   React Query:
   - `useQuery({ queryKey: ['admin-knowledge-documents'], queryFn: adminListDocuments })` for document list
   - `useMutation({ mutationFn: adminUploadDocument, onSuccess: () => queryClient.invalidateQueries({ queryKey: ['admin-knowledge-documents'] }) })` for upload
   - `useMutation({ mutationFn: adminDeleteDocument, onSuccess: () => { setDeletingId(null); queryClient.invalidateQueries({ queryKey: ['admin-knowledge-documents'] }); } })` for delete
   - `useMutation({ mutationFn: adminTestQuery, onSuccess: (data) => { setTestResult(data); setTestError(null); }, onError: (err) => { setTestError((err as Error).message); setTestResult(null); } })` for test

   Handlers (passed as callbacks to micro-modules):
   - `handleUpload(file: File)`: call uploadMutation.mutate(file)
   - `handleDelete(id: string)`: set deletingId, call deleteMutation.mutate(id)
   - `handleTestQuery(query: string)`: call testMutation.mutate(query)

   Layout (using motion variants staggerContainer/staggerItem):
   - Header section: title "IA Global" + subtitle "Gerencie a base de conhecimento da metodologia Iris."
   - `<DocumentUploadArea onUpload={handleUpload} isUploading={uploadMutation.isPending} />`
   - Section label "Documentos" with count badge
   - Loading state: 3 skeleton rows (animate-pulse, same pattern as AdminUsuariosPage)
   - Error state: red error card with message
   - `<DocumentList documents={documents} onDelete={handleDelete} deletingId={deletingId} />`
   - Separator (border-t with margin)
   - `<TestQueryPanel onSubmit={handleTestQuery} result={testResult} isLoading={testMutation.isPending} error={testError} />`
