  <action>
Create `backend/src/routes/stripe/checkout.ts` with a single POST `/checkout-session` endpoint:

1. Import Stripe from 'stripe', instantiate with `process.env.STRIPE_SECRET_KEY` and `apiVersion: '2026-01-28.clover'` (same pattern as stripeProductService.ts).
2. Import `requireAuth` and `AuthenticatedRequest` from `../../middleware/auth.js`.
3. Import `supabaseAdmin` from `../../lib/supabaseAdmin.js`.
4. Import `z` from 'zod'.
5. Create Router, apply `requireAuth` to all routes: `router.use(requireAuth)`.
6. Zod schema: `checkoutSchema = z.object({ price_id: z.string().uuid() })` — price_id is the UUID from stripe_products table, NOT the Stripe price ID.
7. POST `/checkout-session` handler:
   - Parse body with checkoutSchema, return 400 on validation failure.
   - Query `supabaseAdmin.from('stripe_products').select('stripe_price_id, status, billing_interval').eq('id', price_id).single()`.
   - If not found or status !== 'active', return 404 `{ error: 'Plano nao encontrado ou indisponivel' }`.
   - If `stripe_price_id` is null, return 500 `{ error: 'Plano sem preco configurado no Stripe' }`.
   - Determine `mode`: if `billing_interval === 'one_time'` then `'payment'`, else `'subscription'`.
   - Call `stripe.checkout.sessions.create({ mode, line_items: [{ price: product.stripe_price_id, quantity: 1 }], success_url: '${process.env.FRONTEND_URL}/checkout/sucesso?session_id={CHECKOUT_SESSION_ID}', cancel_url: '${process.env.FRONTEND_URL}/planos', client_reference_id: req.userId, metadata: { user_id: req.userId, plan_id: price_id } })`.
   - Return `{ url: session.url }`.
   - Catch Stripe errors: return 502 `{ error: 'Erro ao criar sessao de checkout' }`.
8. Export default router.
  </action>
