```typescript
import { Router, type Request, type Response } from 'express';
import crypto from 'crypto';
import { supabaseAdmin } from '../../lib/supabaseAdmin.js';

const router = Router();

// HMAC-SHA256 verification
function verifyRecallSignature(rawBody: string, signature: string, secret: string): boolean {
  try {
    const expected = crypto.createHmac('sha256', secret).update(rawBody).digest('hex');
    return crypto.timingSafeEqual(Buffer.from(signature), Buffer.from(expected));
  } catch {
    return false;
  }
}
```

**POST /** handler:
1. Extrair raw body: `JSON.stringify(req.body)`
2. Extrair signature: `req.headers['x-recall-signature'] as string`
3. Verificar HMAC com RECALL_WEBHOOK_SECRET. Se invalido E NODE_ENV === 'production', retornar 401 `{ error: 'Invalid signature' }`
4. Parsear body. Esperados eventos:
   - `"transcript.data"`: salvar transcrição
   - `"bot.status_change"`: atualizar status da sessão
5. Para `transcript.data`:
   - Extrair bot_id, transcript.speaker, transcript.words
   - Buscar meeting_sessions por recall_bot_id = bot_id (.single())
   - Se sessão não encontrada, log warning e retornar 200 (evitar retry)
   - Concatenar words[].text em raw_text com espaços
   - Inserir em meeting_transcripts: session_id, speaker, words (jsonb), raw_text, timestamp (now)
   - Retornar 200 `{ ok: true }`
6. Para `bot.status_change`:
   - Mapear status do Recall para status interno: joining -> joining, in_call_not_recording/in_call_recording -> in_call, call_ended -> processing, done -> done, error/fatal -> error
   - Buscar sessão por recall_bot_id
   - Atualizar meeting_sessions.status WHERE status NOT IN ('done', 'error') — ignorar silenciosamente se sessão já está em estado terminal (previne started_at nulo quando evento 'done' chega antes de 'in_call')
   - Se status = 'in_call', setar started_at = now() (somente se started_at IS NULL)
   - Se status = 'done' ou 'error', setar ended_at = now()
   - Retornar 200 `{ ok: true }`
7. Eventos desconhecidos: retornar 200 `{ ok: true, ignored: true }` (nunca 4xx para evitar retries)

Export default router.
