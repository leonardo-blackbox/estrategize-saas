     - `id` uuid DEFAULT gen_random_uuid() PRIMARY KEY
     - `name` text NOT NULL
     - `description` text
     - `stripe_product_id` text UNIQUE
     - `stripe_price_id` text UNIQUE
     - `price_cents` integer NOT NULL CHECK (price_cents >= 0)
     - `credits` integer NOT NULL DEFAULT 0 CHECK (credits >= 0)
     - `billing_interval` text NOT NULL CHECK (billing_interval IN ('month', 'year', 'one_time'))
     - `status` text NOT NULL DEFAULT 'active' CHECK (status IN ('active', 'archived'))
     - `created_at` timestamptz DEFAULT now()
     - `updated_at` timestamptz DEFAULT now()
   - Enable RLS: `ALTER TABLE stripe_products ENABLE ROW LEVEL SECURITY;`
   - RLS policy: admin-only via service_role (no user-level access needed, all access goes through backend admin routes)
   - Add `updated_at` trigger using existing `moddatetime` or manual trigger pattern from codebase
   - Add index on `status` column

2. Create `stripeProductService.ts` (max 150 lines):
   - Import `Stripe` from 'stripe' and instantiate with `process.env.STRIPE_SECRET_KEY`
   - Import `supabaseAdmin` from '../lib/supabaseAdmin.js'
   - Define `StripeProduct` interface matching the table columns
   - Define `CreateProductInput` type: `{ name: string; description?: string; price_cents: number; credits: number; billing_interval: 'month' | 'year' | 'one_time' }`
   - Define `UpdateProductInput` type: `{ name?: string; description?: string; credits?: number; status?: 'active' | 'archived' }`
   - `listProducts()`: SELECT * FROM stripe_products ORDER BY created_at DESC
   - `createProduct(input: CreateProductInput)`:
     a) Call `stripe.products.create({ name: input.name, description: input.description })`
     b) Call `stripe.prices.create({ product: productId, unit_amount: input.price_cents, currency: 'brl', recurring: input.billing_interval !== 'one_time' ? { interval: input.billing_interval } : undefined })`
     c) INSERT into stripe_products with all fields including stripe IDs
     d) Return the DB row
   - `updateProduct(id: string, input: UpdateProductInput)`:
     a) If name changed, update Stripe product name via `stripe.products.update(stripe_product_id, { name })`
     b) UPDATE stripe_products SET ... WHERE id = $id
     c) Return updated row
   - `archiveProduct(id: string)`:
     a) Fetch row to get stripe_product_id
     b) Call `stripe.products.update(stripe_product_id, { active: false })`
     c) UPDATE stripe_products SET status = 'archived' WHERE id = $id
     d) Return updated row
   - All functions wrap in try/catch, throw descriptive errors
